Alarm Conditions with Real-time Response

iBWIM can be set up to detect vehicles that are overloaded and to react in real-time. Alarm limits for Weight, speed, individual axle load and ESAL value can be set.
For each paremter we define 4 states: blue, green, amber and red.
The relationship between parameters and states is shown in online and is defined in a read-only file. The states are defined by their lower limits and are shown in the context of the bridge statistics.
For each of these states there are several methods to respond:
• Variable Messaging systems (VMS) can be used request drivers to continue, or to modify their speed.
• VMS can direct overloaded vehicles off the road.
• CCTV and automatic license plate recognition can be used to identify and trace offending vehicles.
• Enforcement autorities can be informed by email, SMS, or online.
